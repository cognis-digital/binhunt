# Polyglot implementations

Parallel implementations of **binhunt** (Game/desktop binary integrity scanner that fingerprints executables, detects common packers/obfuscators, and diffs against a known-good baseline to catch tampering.) across 9 languages, for reference and cross-language benchmarking.

- `python/` — python implementation
- `go/` — go implementation
- `rust/` — rust implementation
- `typescript/` — typescript implementation
- `c/` — c implementation
- `cpp/` — cpp implementation
- `java/` — java implementation
- `ruby/` — ruby implementation
- `csharp/` — csharp implementation
