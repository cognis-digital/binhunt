binhunt is a command-line tool that checks whether executable files (programs, game clients, installers) have been tampered with or secretly modified. You point it at a file and it tells you if anything looks suspicious — like a packer being used to hide malware, unusually scrambled data inside the program, or changes from a known-good copy you saved earlier. It is aimed at game studios, IT teams, and security researchers who need a quick, scriptable way to catch trojanized or modded binaries before they reach end users.
